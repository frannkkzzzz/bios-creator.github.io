Bug Reports

BiosCreator is a work in progress. Any issues can be reported with details and screenshots via:

Email - cs@leakymosfet.com

Support Forum - https://web.leakymosfet.com/contactus

Feature Requests

You can request a new feature for BiosCreator so we can consider for implementation in the future release of BiosCreator. Should you have one in mind, please feel free to post your request at leakymosfet forum or send them via email.
