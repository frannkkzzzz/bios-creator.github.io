BiosCreator Features

  • Extract stock firmware from various Bios Update Utility file
  • Clean ME/TXE Regions
  • Upgrade/Downgrade ME/TXE Regions
  • Repair/Replace/Extract ME, BIOS and FD Regions
  • Backup windows keys
  • Split EC and System BIOS from a single bin file
  • Extract EC firmware from System BIOS
  • Update/Replace Macbook serial number
  • Decrypt EFI password for Macbook 2011 or earlier
  • Extract stock firmware from Sony VAIO Package Manager
  • Extract firmware from capsule file for laptops with AMD CPUs (Lenovo, Asus, Acer, etc.)
  • Recover HP DMI Info and transfer to new stock firmware (for HP only)
  • Bios Info for HP and Acer
  • Clear NVRAM for UEFI FIRMWARE
  • Master password generator for most brands of laptops
  • Remove/decrypt password for MS Surface Book/Pro 1-6
  • Unlock Supervisor Passwords for Lenovo Thinkpads 4th-8th gen
  • Unlocks Supervisor Passwords for Elitebooks G1(4th gen) to G8(8th gen)
  • Download stock Bios Update Utility from MS Catalog Server
  • Disable On-Board RAM for most laptops
  • Disable enrollment check and enable developer mode in Chromebooks

Supported Models

  • All major laptop and desktop brands are supported.
